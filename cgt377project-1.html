Human Brain in VTK Python

This Python code renders 116 meshes of the human brain. The program allows the user to single out these meshes and toggle different views, such as wireframe. It serves as a base for further development into an anatomy learning tool. The interactive labeling can be used to name parts of the brain. The data is accessible with a premium OsiriX membership.
